# Chess-ai-helper

**Chess AI Assistant** — это интеллектуальный помощник для шахмат, который поможет вам найти лучший ход в любой позиции. Просто расставьте фигуры на доске так, как они стоят в вашей реальной партии, и получите идеальный ход от Stockfish (чемпиона мира по шахматам).

## 🎯 Возможности

- ✅ **Свободная расстановка фигур** — вы сами решаете, где какие фигуры стоят
- ✅ **Визуальная шахматная доска** с красивым оформлением
- ✅ **Перетаскивание фигур мышкой** (drag & drop)
- ✅ **Удаление фигур правой кнопкой мыши**
- ✅ **Переворот доски** для игры с разных сторон
- ✅ **Выбор цвета для ИИ** — можете попросить ИИ сходить как за белых, так и за чёрных
- ✅ **Stockfish на максимальном уровне** — самый сильный шахматный движок в мире
- ✅ **Оценка позиции** после каждого хода ИИ
- ✅ **Звуковое сопровождение** ходов ИИ
- ✅ **Undo / Очистить / Новая игра** — полный контроль над доской

## 🚀 Как пользоваться

1. **Запустите программу** (через `aiasist.py` или `start.bat`)
2. **Расставьте фигуры** на доске так, как они стоят в вашей реальной партии
   - ЛКМ + drag — перетащить фигуру
   - ПКМ — удалить фигуру с клетки
3. **Выберите цвет**, за который должен сходить ИИ (⚪ Белых / ⚫ Чёрных)
4. **Нажмите кнопку** «🤖 СДЕЛАТЬ ХОД ИИ»
5. **Получите идеальный ход** с оценкой позиции!

## 🛠 Управление

| Действие | Описание |
|----------|----------|
| 🖱️ ЛКМ + drag | Перетащить фигуру |
| 🖱️ ПКМ | Удалить фигуру |
| 🔄 Перевернуть | Повернуть доску |
| ↩️ Undo | Отменить последнее действие |
| 🗑️ Очистить | Очистить всю доску |
| 🔄 Новая игра | Вернуть начальную расстановку |
| 🤖 СДЕЛАТЬ ХОД ИИ | Запросить ход у Stockfish |

## 📋 Требования

- **Python** 3.7 или выше
- **Stockfish** (скачать с [официального сайта](https://stockfishchess.org/download/))
- **Библиотеки**: `chess`, `stockfish`, `playsound`

## 💿 Установка

### Вариант 1: Через батник (рекомендуется)

1. Скачайте все файлы из репозитория
2. Запустите `start.bat`
3. Батник автоматически:
   - Проверит наличие Python
   - Установит все необходимые библиотеки
   - Проверит наличие Stockfish
   - Запустит программу

### Вариант 2: Вручную

```bash
# Установка зависимостей
pip install chess stockfish playsound

# Скачайте Stockfish с официального сайта
# Переименуйте в stockfish.exe и положите в папку с программой

# Запуск
python aiasist.py
